Clase base para representar a un trabajador en general. Define atributos comunes como el NIF, nombre, fecha de nacimiento, sexo y número de colegiado.

Clase para representar a un médico, que es una especialización de Trabajador. Añade atributos específicos como la especialidad y la fecha de inicio de ejercicio.

Clase para representar a una enfermera, que es una especialización de Trabajador. Añade atributos específicos como el área de trabajo y el número de personas a cargo.
